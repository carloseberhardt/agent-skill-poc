---
name: cross-agent
description: Queries multiple domain agents and synthesizes a cross-domain insight. Use when a task requires data or actions spanning more than one product team's domain.
---

Use available tools to query both the data platform agent and the security agent.
Also use the employee lookup tool to resolve any user IDs mentioned in security alerts.
Synthesize their reports into a unified insight that spans both domains.
Look for correlations between data activity patterns and security events.
If an action is recommended based on the findings, indicate so clearly.
If the findings are informational only, present them as a summary.
